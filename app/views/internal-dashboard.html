OVERVIEW
  • Produced a seamless internal dashboard, displaying relevant statistics from recent quarters.
  • Thoroughly researched data extraction techniques; tested and employed XHR Requests, Fetch API, jQuery AJAX requests, and CORS requests to withdraw and manipulate data from a variety of sources.
  • Utilized Data-AOS libraries, CanvasJS, and jQuery to maximize UI/UX of the application.
    • SUMMARY
    The Brantford Police Service required an internal dashboard that would report back statistics regarding the most recent quarters for the service. I experimented using XHR requests, Fetch API, and jQuery AJAX requests to obtain the data and display it in a graphical analysis on the webpage. I utilized PHP and MySQL to create a user authentication system so that an individual could manage the dashboard internally.
