OVERVIEW
  • Developed a priority queue data structure using PHP and MySQL to create the turn-based system that would assign the duties to the officers.
  • Utilized a WAMP stack to configure the application, explored CRON tabs and MySQL Events to send email alerts to users when duties were posted.
  • Created a PHP script that would automate the mailing of duties to assigned officers.
  • Used PHPSpreadSheet to export annual reports regarding invoices in a CSV format.
    • SUMMARY
    The paid duty project was the most extensive projects with the Brantford Police Service. I was tasked with making a web application that allowed administration to post Paid Duties online for officers to see and apply for, a CRUD application. I used MySQL databases to store duties posted, users who applied to certain duties, and users. I interacted with the database using PHP, and experimented with CRON tabs and MySQL events to automatically assign users for duties. I also utilized mail-servers to automatically send alerts to users when updates were made to the application.
